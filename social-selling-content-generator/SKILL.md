# Social Selling Content Generator

Generate a 30-day LinkedIn content calendar and engagement playbook that positions the user as an authority and attracts inbound B2B leads.

## Contents
- references/output-template.md - Calendar structure and coverage requirements
- references/examples.md - Worked post examples for each post type
- references/comment-strategy.md - Turning comments into conversations
- references/persona-templates.md - Topic banks by buyer persona
- references/engagement-tactics.md - Timing, formatting, and quick-start plan

## Workflow

1. Capture inputs: target audience (job titles/roles), what the user sells, content goal (attract/educate/engage), and posting frequency.
2. Define the personal brand position as one sentence and select 4-5 content pillars: Industry Insights, Problem/Solution, Personal Stories, How-To/Educational, Social Proof.
3. Plan post-type variety across the calendar: question, story, list, contrarian, data, and case-study posts.
4. Seed topics from the buyer persona using references/persona-templates.md.
5. Generate all 30 posts following references/output-template.md. For each post write a hook, full body in the user's voice, a "Why This Works" rationale, and a best-time-to-post. Match the style shown in references/examples.md without copying it.
6. Append the comment strategy from references/comment-strategy.md so the user can convert engagement into leads.
7. Append engagement tactics and the 4-week quick-start plan from references/engagement-tactics.md.

## Rules
- Write in the user's authentic voice; never pitch directly in posts.
- Use specific numbers and concrete examples; no placeholders in delivered posts.
- Recommend moving qualified conversations off-platform (DM to call to meeting).
- Set expectations: social selling builds momentum over 3-6 months.
